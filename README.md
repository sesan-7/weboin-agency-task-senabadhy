# Design Agency Homepage – Next.js 13

This project is a modern design agency homepage built using Next.js 13 (App Router).

## Features
- Hero Section
- Services Section
- Portfolio Grid
- Contact Form
- Fully Responsive
- Built with Tailwind CSS
- Framer Motion animations
- Optimized with Next/Image

## Tech Stack
- Next.js 13+
- TypeScript
- Tailwind CSS
- Framer Motion

## Setup Instructions

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd weboin
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

### 4. Open the application
Open [http://localhost:3000](http://localhost:3000) in your browser.


##  Additional Features & Project Assumptions


To make this project production-ready and scalable for real-world usage, the following improvements can be implemented:

Admin Dashboard with secure authentication
- Dynamic content management for Services & Portfolio
- Database integration (e.g., PostgreSQL / MongoDB)
- CMS integration (Sanity / Strapi)
